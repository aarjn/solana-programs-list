# 🎲 Dice Pool Game

A decentralized **Dice Pool Betting Game** built using smart contracts, where users participate in a shared pool and win rewards based on correct dice predictions.

---

## 🚀 Overview

Dice Pool Game allows users to:
- Join a betting pool with a fixed base amount  
- Predict the outcome of a dice roll (1–6)  
- Win rewards if their prediction is correct  

The game ensures **fairness, transparency, and on-chain execution**.

---

## 🧩 Features

- 🏗️ **Pool Creation**
  - Owner creates a pool with a predefined base bet  

- 👥 **User Participation**
  - Users join the pool with equal stake  

- 🎲 **Dice Betting**
  - Users place bets by choosing a dice outcome  

- 🎯 **Result Reveal**
  - Owner reveals the dice result  

- 🏆 **Reward Distribution**
  - Winners share the total pool proportionally  

- 💸 **Secure Withdrawals**
  - Winners can withdraw their rewards safely  

---

## 🔄 Game Flow

```text
1. Owner creates pool
2. Users join pool with base bet
3. Users select dice number (1–6)
4. Dice result is revealed
5. Winners are determined
6. Rewards distributed
7. Winners withdraw funds
